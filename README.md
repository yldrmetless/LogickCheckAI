LogicCheck AI

🚀 Key Features
AI-Powered Analysis: Leverages advanced AI models to evaluate the logical consistency and market viability of business ideas.

Automated Business Plans: Generates detailed business structures and strategic steps in seconds.

Secure User Profiles: Managed via JWT authentication for private project storage.

Modern Dashboard: A clean UI/UX for managing, updating, and tracking analyzed reports.

🏗️ Technical Architecture
The project follows a decoupled, full-stack architecture with a focus on scalability and modern development practices.

Backend
Framework: Django REST Framework (DRF)

Database: PostgreSQL (Hosted on Neon.tech)

DevOps: Dockerized environment for consistent local and production builds.

Testing: Comprehensive test suite with 42+ automated tests using Pytest.

Frontend
Framework: Next.js (React)

Styling: Tailwind CSS

State Management: Modern React hooks and efficient API integration.
